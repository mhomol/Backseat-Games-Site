Getting started

  • Joining is free. Tap Join a Game and pick the host’s session on the same Wi‑Fi or Personal Hotspot.
  • Hosting requires a one-time in-app purchase on iPhone. Passengers never pay.
  • No internet needed on the trip after the host unlock — phones connect locally in the car.

Common issues

  • No nearby games? Confirm the host started a session. Try connecting all phones to the same Personal Hotspot.
  • Restore hosting purchase? In the app: Settings → Restore purchases (same Apple ID as the original purchase).
  • Microphone is only used for optional Sign Game recordings.

Pricing

The app is free to download. Joining games is free. Starting/hosting a session requires a one-time $1.99 in-app purchase.
